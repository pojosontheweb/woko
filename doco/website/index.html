POJOs on the Web!

Tired of heavyweight, unproductive and boring web development in Java ?
Woko is a Full-Stack, Domain-Driven Web Framework that aims to put your POJOs on the Web for nothing (or almost !).

Get started »

Persistence & Search

No-config persistence and full text search services for your POJOs, with built-in support for Hibernate and Compass.
No more DAO hassles and Session management : all you need to worry about is your Domain Objects.

Users & Roles

First-class support for roles and users. Authentication, user management, GUI profiling...
It's all built-in, and completely pluggable if you need to include your own user management system.

MVC & Object Renderer

Base MVC plumbing (URLs, data binding & validation, controllers, templating etc.), and facet-based Object Renderer : Woko can display your objects without you writing code for it !

JSON & Javascript APIs

Out of the box JSON/HTTP services for your application, and a JavaScript API for accessing your Domain Objects and back-end, right in the page.
