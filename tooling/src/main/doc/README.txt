woko - The woko developers' tool
------------------------------------------------------------------------------

Utility that allows to create woko project
Full doc, bug reports, contrib, etc. at :

    https://github.com/vankeisb/woko2


Installation
------------


Set the environment variable WOKO_HOME to the extracted woko folder, e.g. :

  bash$ export WOKO_HOME=/Users/johndoe/tools/woko


On unix, set execution permissions on the woko script :

  bash$ chmod +x $WOKO_HOME/bin/woko


Add woko to your PATH environment variable :

  bash$ export PATH=$PATH:$WOKO_HOME/bin


You're ready to go ! To try your install, cd in you projects' folder and type woko :

  bash$ cd /Projects
  bash$ woko


... you should see the usage message, which means everything should be ok !
